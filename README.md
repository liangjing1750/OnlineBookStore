**DDD案例**

**初级**
- (1)从面向对象到战术设计 (聚合、实体)
- (2)战术设计之值对象(领域层的实体级别)
- (3)战术设计之资源库(数据库层)
- (4)战术设计之应用服务 (应用层)
- (5)战术设计之分层架构 (完整服务)
  
**中级**
- (1)战术设计之事件驱动 (状态机)
- (2)战术设计之六边形架构 (低耦合)
- (3)战术设计之领域服务(多聚合)
- (4)战略设计之领域、限界上下文
- (5)战略设计之事件风暴 (四色建模)
- (6)DDD之整洁代码(面向对象、SOLID、设计模式、重构与坏味道)
- (7)DDD之整洁架构(架构演进)

**高级**
- (1)DDD与微服务: 分布式事务
- (2)DDD与行为驱动开发BDD
- (3)DDD与敏捷、DevOps 
- (4)DDD与企业架构
